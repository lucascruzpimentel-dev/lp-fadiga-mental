05 — Motion & Interaction

Animations, hovers, easing

All keyframes are inlined verbatim from index.html. Hover states use Tailwind transition-colors / transition-all with the default 150 ms duration unless noted.

3D card tilt

.parent:hover .card

Hover the wrapper. rotate3d(1,1,0,30deg) with layered shadow on the child.

Hover me 
.parent:hover .card {
  transform: rotate3d(1,1,0,30deg);
  box-shadow:
    rgba(5,71,17,0.3) 30px 50px 25px -40px,
    rgba(5,71,17,0.1) 0 25px 30px 0;
}

Each layer of the hero card animates in on its own timing. Below: the keyframes used and their default delays.
