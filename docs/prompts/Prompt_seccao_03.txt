## CARD COM BORDA ANIMADA (coluna esquerda)

Essa é a parte mais importante da seção. 
Implemente com precisão.

O card representa um certificado. 
Use proporção aproximada de A4 paisagem (aspect-ratio: 1.41 / 1).

**Estrutura do efeito de borda girante:**

Crie um elemento wrapper com position: relative e padding: 2px. 
Esse wrapper recebe um pseudo-elemento ::before que ocupa todo 
o espaço (inset: 0) e exibe um conic-gradient que gira 
continuamente, criando a ilusão de uma luz percorrendo a borda.

Declare a propriedade customizada no CSS:

  @property --angle {
    syntax: '<angle>';
    initial-value: 0deg;
    inherits: false;
  }

O ::before deve ter:

  background: conic-gradient(
    from var(--angle),
    transparent 0deg,
    transparent 290deg,
    rgba(37, 99, 235, 0.0) 300deg,
    rgba(59, 130, 246, 0.7) 328deg,
    rgba(255, 255, 255, 1.0) 343deg,
    rgba(59, 130, 246, 0.7) 357deg,
    transparent 360deg
  );
  animation: border-spin 4s linear infinite;

  @keyframes border-spin {
    to { --angle: 360deg; }
  }

O ponto de luz branca ocupa apenas ~15 graus — 
deve ser sutil, não chamativo. O restante da borda 
é transparente, não azul contínuo.

O card interno (filho do wrapper) deve ter:
  - background: rgba(13, 16, 32, 0.97)
  - border-radius: 14px (wrapper com 16px)
  - padding: 40px
  - display flex, coluna, justify-content center, align center
  - gap entre elementos internos: 16px

Conteúdo interno do card (placeholder elegante do certificado):
  - Logo assets/images/logo.png com height 28px, centralizada

## ANIMAÇÕES DE ENTRADA (Scroll Reveal)

Usar IntersectionObserver com threshold: 0.15.
Iniciar todos os elementos com opacity: 0.

Quando a seção entrar no viewport, disparar em sequência:

1. Card wrapper:
   opacity 0 → 1, translateX(-28px) → 0
   transition: 0.85s ease, delay 0s

2. Label + headline + descrição (coluna direita):
   opacity 0 → 1, translateX(28px) → 0
   transition: 0.85s ease, delay: 0.15s

3. Bullets individualmente com stagger:
   opacity 0 → 1, translateY(12px) → 0
   transition: 0.5s ease
   delay: bullet 01 = 0.3s, 02 = 0.42s, 03 = 0.54s

Adicionar o IntersectionObserver ao bloco de scripts 
já existente no final do body. Não criar script separado.
